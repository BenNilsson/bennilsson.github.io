A Broken Village

A Broken Village is a 2D top-down Stardew Valley inspired game created for the Global Game Jam 2020 where the player must repair his houses that are getting attacked by enemies. My role within this game was gameplay programming, UI design/implementation, and level building. We were a total of 3 people working on this game.

Features Included

  • Inventory
  • Repairing Buildings
  • Combat
  • Gathering
