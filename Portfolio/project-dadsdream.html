A Broken Village

With only 8 hours left of the Global Game Jam 2018 and me scratching my project, I decided to work on Dad's Dream: A top-down 2D collection game where you, the son, is asked to collect several items spawning around the room by your father. My aim with this project was to make a silly yet fun game and it had a great response from many participants at the game jam site. A true focus on gameplay.

Features Included

  • Collecting Items
  • Retrieving Items
