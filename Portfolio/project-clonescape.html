CloneScape

This project was made for my 2nd year of university (level 5) for the first semester of the module known as Game Engine Programming. We were tasked to create a game in Unity.

Working on CloneScape was one of my more exiting projects. I spent a total of 6 weeks developing this tiny game. I found myself taking inspiration from the original version (Old School Runescape) and attempting to implement similar features. Overall, the gameflow is really nice and I would love to continue working on similar titles.

Features Included

  • Quest System
  • Combat System
  • Inventory
  • Simple AI
  • Event-based Programming

See the video below for a snippet of gameplay

Learning Outcomes

This project was one of the first projects where I attempted to connect bigger systems together. My most proud achievement is the quest system. It is a system that can easily be expanded to cover whatever is necessary. We were taught a lot about the Unity engine this semester and I managed to take that knowledge and put it into practice.

Tools Used

  • Unity 2020.1.4f1
  • Microsoft Visual Studio 2019
  • Blender
