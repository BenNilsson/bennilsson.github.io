Orcs & Dragons TD

This project was made for my 2nd year of university (level 5) for the first semester of the module known as Games Development. We were tasked to create a tower defense game and create a marketing plan for it.

Orcs & Dragons TD is a tower defense game inspired by an old Warcraft III custom map known as Wintermaul One. In this multiplayer game, you defend a house from orcs and dragons (hence the creative game). The game supports up to 4 players. Keep in mind that this is not a PvP-based game. It is rather an experience that you can have alongside others. The game has 4 types of towers which can be placed on a grid. The placement of the towers creates a maze for the enemies to walk around. The player must defeat 30 waves to win the game. If 20 enemies make it to the house, the player will lose.

Features Included

  • Photon 2 Multiplayer Support
  • Grid-based Building System
  • Enemy Pathfinding (A*)
  • Tower Selection & Targeting
  • Event-based Programming

Learning Outcomes

This game was my first attempt at creating a multiplayer game. With lots of bug fixing, I was able to create a really fun tower defense game based on my favourite content from older games and re-create it. This was also the first project where I learned how to use events within Unity/C#.
