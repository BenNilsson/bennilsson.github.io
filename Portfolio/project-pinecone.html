Pinecone Networking

Pinecone Networking is a lightweight C# networking library designed for use in multiplayer games, made for the Unity game engine.

This project was created as a university project for the module Games Technology Research Project, commonly referred to as Final Year Project.

This page will be updated further as the development of this project continues.
