Intro

Hi, my name is Benjamin Nilsson. I am a games developer enthusiast who decided to make a major change in life and move out of country to persue my passion for creating games. I am currently a student at Staffordshire University, studying Computer Games Development full time.

I spend the majority of my free time thinking of what projects to work on next, exploring multiple fields within the games industry, researching and expanding my knowledge within the field of games programming but most importantly, I am a games addict who loves a good title. My true dream is quite simple: I wish to create a game which I personally would be able to play several hours a day without getting bored. I believe that if a games developer can achieve that, they must have done well at some point.

Thank you for checking out my portfolio! Consider having a look at some of My Projects!

Educational Work

Pacman Clone

During the first semester of our first year at university (level 4), we were assessed to create a Pacman clone in which I attempted be as close to the original as I could get without any prior experience in C++ whatsoever. As a first project, I am extremely proud of how far I get and it is definently a proof to myself of where my passion lies at. I am incredibly excited to continue my development within games

Game Jams

Steve and Dave's
Grassland Adventures

Other than being the first ever team-work based project I created, Steve and Dave's Grassland Adventures was a really fun project to work on. We set a scope too big and had to drop plans but ended up creating a functioning side-scrolling RPG with a Combat System, Enemy AI, Dialog System and much more. My contribution towards this project was all of the programming aspects done using C# as well as teaching the engine to co-member of the team.

Tools & Software used: Unity 2018

Vix

Vix is a 3D exploratin-based game set in a city's alleyways. The project was my first attempt at using the Unreal Engine. Coming in with no prior experience within the engine, I was tasked to work on small Blue Print related tasks as well as outlining the level, building the level as well as cinematics. This was truely an amazing experience as it expanded my horizon within game engines.

Tools & Software used: Unreal Engine 4.23.1

View Project Here

Dad's Dream

With only 8 hours left of the Global Game Jam 2018 and me scratching my project, I decided to work on Dad's Dream: A top-down 2D collection game where you, the son, is asked to collect several items spawning around the room by your father. My aim with this project was to make a silly yet fun game and it had great response from many participants at the game jam site. A true focus on gameplay.

Tools & Software used: Unity 2018
